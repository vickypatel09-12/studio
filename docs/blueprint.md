# **App Name**: Bachat Bank ERP

## Core Features:

- Dashboard Overview: Summarizes key metrics like total deposits, outstanding loans, and financial health.
- Monthly Deposit Entry: Allows manual entry of monthly deposits (Sr. No., Customer Name, Deposit Amount, Payment Method).
- Loan & Interest Management: Tracks loans, changes in loan amounts, and calculates interest (Sr. No., Customer Name, Carry Fwd Loan, New Loan, Interest).
- Interest Calculation Tool: AI-powered tool that calculates customer interest due on loans for entry into the system.
- Customer Management: Creates, edits, and deletes customer profiles with import/export CSV functionality.
- Settings Configuration: Configures application settings like interest rates and monthly session start/end.
- Report Generation: Downloads financial reports in formats like CSV and PDF.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6), evoking trust and stability in financial management.
- Background color: Light blue (#E3F2FD), provides a clean and airy backdrop.
- Accent color: Soft green (#81C784), symbolizes growth and positive financial outcomes.
- Body and headline font: 'PT Sans', a modern sans-serif, for clear and accessible information presentation. Support gujarati font.
- Use minimalist, clear icons to represent different functions and data points within the ERP.
- A clean, tabular layout with clear spacing to organize the data-heavy content effectively.
- Subtle transitions and animations to provide feedback on user interactions without being distracting.
